## 📖 简介

Vue Flow 流程图编辑器是一个专业的可视化流程设计工具，提供了直观的拖拽界面和丰富的节点类型，适用于业务流程设计、工作流编排、状态机设计等场景。

### ✨ 亮点

- 🎨 **美观的界面** - 现代化的 UI 设计，支持暗黑模式
- 🖱️ **直观的操作** - 拖拽式操作，零学习成本
- 🔧 **高度可定制** - 支持自定义节点类型和样式
- 📱 **响应式设计** - 适配各种屏幕尺寸
- ⚡ **高性能** - 优化的渲染性能，支持大型流程图
- 🛡️ **类型安全** - 完整的 TypeScript 支持
- 📦 **开箱即用** - 无需复杂配置，快速上手

## 🚀 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 访问页面

在浏览器中打开：`http://localhost:你的端口/test/vueflow`

### 5分钟教程

1. **添加节点** - 从底部面板拖拽节点到画布
2. **连接节点** - 拖动节点的连接点创建连接
3. **编辑属性** - 点击节点，在右侧面板编辑
4. **保存流程** - 点击"导出数据"按钮

详细教程请查看 [快速开始指南](./QUICKSTART.md)

## 🎯 功能特性

### 核心功能

#### 🎨 丰富的节点类型

- **开始节点** - 流程起点，圆角矩形，绿色
- **结束节点** - 流程终点，圆角矩形，红色
- **流程节点** - 普通处理步骤，矩形，蓝色
- **判断节点** - 条件分支，菱形，橙色
- **并行节点** - 并行处理，矩形，灰色
- **自定义节点** - 可扩展的自定义节点

#### 🖱️ 拖拽操作

- ✅ 从节点面板拖拽节点到画布
- ✅ 在画布上拖动节点调整位置
- ✅ 拖拽节点到面板删除
- ✅ 拖动连接点创建连接

#### ✏️ 编辑功能

- ✅ 修改节点名称和描述
- ✅ 自定义节点颜色
- ✅ 调整节点位置坐标
- ✅ 设置节点连接状态
- ✅ 添加连接标签

#### 🎛️ 视图控制

- ✅ 缩放画布（20% - 400%）
- ✅ 平移画布
- ✅ 重置视图
- ✅ 适应画布
- ✅ 小地图导航
- ✅ 暗黑模式

#### 💾 数据管理

- ✅ 导出流程数据（JSON）
- ✅ 清空画布
- ✅ 删除节点/边
